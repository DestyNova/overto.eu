Benben

Today I was absolutely delighted to stumble upon a new command-line VGM (Video Game Music file) player called Benben. The author mentioned it in the Crystal programming language forum along with the underlying VGM playback library YunoSynth.

Benben is distributed as an executable AppImage file which worked perfectly out of the box on my Pop!_OS Linux system. Here’s a video of it playing the soundtrack to the PC-98 game "Rusty":

Give it a try! You can find a collection of files to test it on here.
